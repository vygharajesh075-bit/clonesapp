## 🚀 What is This?

A growing **frontend clone universe** where real-world apps are recreated from scratch using pure frontend skills.

Think of it as:
> 🧠 “Reverse engineering popular UI into learning-powered builds”

---

## 🧩 Project Collection Inside

This repo is not just one project. It’s a **bundle of UI clones** like:

- 🎬 Movie App Clone (Netflix-style UI)
- 🌤️ Weather Dashboard Clone
- 🎵 Music Player Clone
- 📝 Notes / Todo App Clone
- 📊 Mini Analytics Dashboard
- 🤖 AI Chat UI Clone (frontend only)

Each clone focuses on **layout, interaction, and real UI behavior replication**.
